Programming – making computers do what we want them to
3 categories of programming languages
Low-level programing – Language that is closer to the hardware of a computer
High-level programming – Languages that are easier to understand and write
Script – Languages that turn into machine code upon running


JavaScript – High level language that is compiled at run time
Also a dynamic langue, meaning that elements of the program can while its running
So, a program can run things in the background like type or spell checking

HTML5 – The dominant programming language for the web
JavaScript is a key feature in using html5 to its fullest potential

Unobtrusive JavaScript
Originally, java script was directly inputted into the html code
This made it much harder to read, write, and understand
A script tag can be used in the html to keep all of the JavaScript in one place
Making it much easier to understand and write
A link to a JavaScript file can be used script src=”main.js”
Takes all java code out of the html file creating an organized space for JavaScript code and allows for large functions to be written


JavaScript Arrays
Used to store multiple values in a single variable
To create an array:
var array_name = [values]
Can also use key word new to create an array:
Var array_name = new Array(values)
This way yields the same result as the other but is less simple, so using the other way is best
Can access the elements in an array using the index number
For example, [0] would point to the first element
Can change specific elements in array’s using the index
Array_name[0] = “new value”
Full arrays can be acsessed by using the full name of the array


Arrays properties and methods
There are various properties and methods for using arrays in JavaScript
Length property
Returns the number of elements in the array
Sort() property
Sorts the array
Push() property
Adds new elements to the array
These are just a few examples, there are more


JavaScript Loops
For loop
Loops through a block of code a number of times
Syntax:
for (statement 1; statement 2; statement 3) { // code block to be executed }
For/in loop
Loops through the properties of an object
Syntax:
for (x in object) { }
For/of loop
Loops through the values of an inerrable object
Syntax:
for (variable of iterable) { }
While loop
Loops through a block of code when a certain condition is true
Syntax:
while (condition) { }
Do/while loop
Loops through a block of code when certain condition is true (similar to while)
Syntax:
Do { } While (condition)

JavaScript Functions
A function is a block of code designed to perform a particular task.
Use the function key word to define a function: function my function() { }
The code inside a function will only execute when something invokes (calls) it
The return statement is used to return a value when the function is called
Local variables can be described in the scope of the function and only used in that scope
Functions offer a way to reuse code multiple times
